# CallingApp

This is an Android application that simulates an incoming call and active call screen.

## Features

* Incoming call screen
* Accept and reject call
* Call timer
* Mute button (UI only)
* Speaker button (UI only)

## Tech Used

* Kotlin
* Android Studio
* XML layouts

## Download APK

Download and install the app using the APK file:

app-debug.apk

## Project Structure

* app/ → main application code
* res/ → layouts and UI
* AndroidManifest.xml → app configuration
